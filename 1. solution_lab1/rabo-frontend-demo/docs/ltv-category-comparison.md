# Does my interest rate change based on my savings?

## Example
Yes! The more you can save, the better your rate.

### Prompt

```
What interest rates does Rabobank offer depending on how much I can save?
Show me the rates for 10 years fixed in these situations:

- With the best protection option (about 10% saved)
- If I've saved 30%
- If I've saved between 10-30%
- If I've saved less than 10%
```

### What you get back
You'll see how rates change based on your savings.

### How much extra will it cost?
```
I'm considering three options:
1. €30,000 saved (€300,000 house)
2. €75,000 saved (€300,000 house)
3. €105,000 saved (€300,000 house)

How much extra interest do I pay in a year if I save less?
```

### Total interest difference
```
How much more interest would I pay in total if I save less?
Compare 30 years of payments on a €300,000 house.
```
