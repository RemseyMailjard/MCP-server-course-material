# Accessibility Guidance: Inclusive Layouts

Summary: Build layouts that stay understandable when zoomed, resized, simplified, or assisted by technology, so the design keeps its balance under pressure.

## Guidelines
- Let content reflow cleanly at higher zoom levels and narrower widths without clipping actions or instructions.
- Preserve spacing and grouping so related information stays together when components stack on smaller screens.
- Do not remove useful context just to make a layout look minimal; clarity should win over decoration.
- Test composed surfaces with keyboard use, zoom, and screen reader basics before calling the demo finished.

## Example Prompt
Explain how to keep the story page layout accessible when the feature-panel stacks on mobile.

## Joke
Responsive layouts are a lot like bicycles in the Netherlands: they work best when there is room for everyone.
